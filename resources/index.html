Resources

Resources to learn programming.

I want to learn programming. Where do I start?

The best way to start to learn programming varies from person to person. Some people prefer to learn from the ground-up through reading introductory and foundation material, while others might benefit more from a direct hands-on approach by starting a coding project.

There's no right or wrong answer, but here is a collection of resources we have found helpful in the past:

Advice on Starting to Learn Programming

  • Teach Yourself Computer Science - a comprehensive collection of resources, more geared towards those pursuing software engineering as a proper career than as a hobby, but very helpful as a resource guide.

Books

  • Automate the Boring Stuff with Python - a popular book geared at using the Python programming language for pragmatic tasks in a hands-on approach.

Online Courses

  • MIT OpenCourseWare: Introduction to Computer Science and Programming in Python
  • Harvard CS50: Introduction to Computer Science

Practise Problems

  • Rosalind - a huge collection of bioinformatics problems (ranging from very easy to very hard), helpful to practise coding skills.
  • Project Euler - a collection of maths problems with computational solutions.
